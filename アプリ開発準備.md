# アプリ開発準備

## はじめに

このテキストは、

- VS Code
- GitHub
- Python仮想環境(.venv)

を組み合わせて、安全かつ効率的にPython開発を行うためのガイドです。

特に、
- 複数PC運用
- GitHubをマスタ管理にする
- VS Code中心の開発
- pandasなど外部ライブラリ利用
を前提にしています。

---

```text

├── README.md
├── main.py

```


# STEP 3 仮想環境作成

コマンドプロンプトで：

```bash
python -m venv .venv
```

成功すると：

```text
.venv/
```

フォルダ生成。

# STEP 4 pandasインストール
pandas以外の必要モジュールもここでインストールしておくこと。
```bash
pip install pandas
```

---

# STEP 5 サンプルプログラム作成

main.pyを作成する

---

# STEP 6 動作確認

```bash
python main.py
```

---

# STEP 7 requirements.txt作成

```bash
pip freeze > requirements.txt
```

生成例：

```text
pandas==2.2.3
numpy==2.1.2
python-dateutil==2.9.0.post0
pytz==2025.2
tzdata==2025.2
```

---

# requirements.txtとは？

「この環境に必要なライブラリ一覧」です。
別PCで：

```bash
pip install -r requirements.txt
```

することで同じ環境を再現できます。

---

# STEP 8 .gitignore作成

## .gitignoreとは？
Gitに「これらはGitHubへ送らないで」と指示するファイル。

```gitignore
# 仮想環境
.venv/

# Pythonキャッシュ
__pycache__/
*.pyc

# 環境変数
.env

# SQLite
*.db
```

---


これで：

```text
.git/
```

# STEP 12 別PCでの作業開始

## ① GitHubから取得

```bash
git clone https://github.com/USERNAME/sample_app.git
```

## ② フォルダへ移動

```bash
cd sample_app
```

## ③ 仮想環境作成

```bash
python -m venv .venv
```

## ④ 仮想環境有効化

```bash
.venv\\Scripts\\activate
```

## ⑤ ライブラリインストール

```bash
pip install -r requirements.txt
```

## ⑥ VS Codeで開く
